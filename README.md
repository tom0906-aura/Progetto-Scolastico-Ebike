# Progetto-Scolastico-Ebike
Questo codice fatto per un progetto scolastico in C++ gestisce in modo automatico il noleggio di un numero di biciclette elettriche. 
Il programma acquisisce la posizione dell'utente e mostra tutte le e-bike disponibili ordinate da quella più vicina a quella più lontana, calcolando per ognuna la distanza esatta e la batteria rimasta. 
Dopo aver scelto una bici tramite il suo numero identificativo (ID), il software controlla se l'autonomia è sufficiente per il viaggio richiesto. 
Infine, calcola il tempo necessario comprensivo di un ritardo casuale dovuto al traffico, stima il costo totale in Euro e aggiorna i chilometri residui della bicicletta scaricando la batteria.
